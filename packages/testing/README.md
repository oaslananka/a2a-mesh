# a2a-mesh-testing

Testing helpers for building fast in-process A2A integration tests.

This package stays in the monorepo for advanced testing workflows, but it is not part of the primary public npm surface for the first launch.

## Includes

- `A2ATestServer`
- `MockA2AClient`
- reusable fixtures
- `toHaveCompletedTask` matcher

## Install

```bash
npm install -D a2a-mesh-testing
```
