# `a2a-mesh-grpc`

The gRPC package provides an experimental transport binding for A2A-compatible systems that need strongly-defined service contracts and non-HTTP integration points.

This package is kept in the repository but is **not part of the first public npm launch wave**. Treat
it as an in-repo experimental transport companion until a later npm release.

It ships with:

- `GrpcServer`
- `GrpcClient`
- `proto/a2a.proto`
